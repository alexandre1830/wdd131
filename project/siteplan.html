Hey, Teacher!

"Hey, Teacher!" is the name of my personal project (and work) of private English classes

Site Purpose

The purpose of this site is to provide information about my English teaching services, including class offerings and contact details.

Scenarios

Some potential scenarios and questions for using this site include:

  • What is the method used to teach English?
  • What are the available class formats (e.g., online, in-person)?
  • How can this course help me improve my English skills?
  • What are the main differences between the traditional English courses?
  • How can I contact the teacher for more information or to schedule a class?

Color Schema

The color scheme for the "Hey, Teacher!" website is already in use on other materials:

  • blue: #032d6f; (header and titles)
  • red: #a60404; (subtitles and buttons)
  • lightblue: #8095b6; (details)
  • lightred: #d18180; (details)
  • white: #ffffff; (some text, icons and details)
  • black: #000000; (main text)

The color scheme is designed to be visually appealing and to create a cohesive look across all materials.

Typography

The typography for the "Hey, Teacher!" website is also an important aspect of the design:

  • Headings: 'Josefin Sans', sans-serif
  • Body text: 'Roboto', sans-serif

Wireframe

The wireframe for the "Hey, Teacher!" website is designed to be simple and user-friendly, with a clear navigation structure and easy access to important information. The homepage features a header with the logo and navigation menu, followed by a main section with an introduction to the project and its purpose.
